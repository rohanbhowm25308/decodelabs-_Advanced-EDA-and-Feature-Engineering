#  Advanced EDA & Feature Engineering on E-Commerce Sales Data

##  Overview

Data is only as valuable as its quality. This project focuses on transforming raw e-commerce sales data into a clean, structured, and machine-learning-ready dataset through comprehensive Exploratory Data Analysis (EDA) and Feature Engineering.

By applying statistical data cleaning techniques, outlier treatment, and intelligent feature creation, this project demonstrates how real-world business data can be prepared for advanced analytics and predictive modeling.

---

##  Project Objectives

* Analyze and understand raw e-commerce transaction data.
* Identify and handle missing values efficiently.
* Detect and treat outliers using statistical methods.
* Generate meaningful visual insights through EDA.
* Engineer new features to enhance predictive capability.
* Build a clean dataset suitable for Machine Learning applications.

---

##  Data Cleaning Process

### Missing Value Treatment

Missing values were identified and handled using appropriate statistical techniques to maintain data integrity.

### Duplicate Removal

Duplicate records were detected and removed to ensure dataset consistency.

### Outlier Detection

Outliers were identified and treated using the **Interquartile Range (IQR)** method, improving overall data reliability.

---

##  Feature Engineering

To enhance the analytical value of the dataset, several new features were created:

### Average Item Price

Provides insight into the average cost per purchased item.

### Cart Value

Represents the estimated monetary value of items present in a customer's cart.

### Coupon Usage Indicator

Identifies whether a customer utilized promotional discounts.

### Order Month

Extracted from transaction dates to support seasonal trend analysis.
